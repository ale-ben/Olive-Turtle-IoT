char * getTimestamp ( )

Save the timestamp string in the timestamp parameter. Timestamp string is defined as:

  • "[YY-MM-DD_HH:MM:SS]\0" in normal conditions
  • "[YY-MM-DD_HH:MM:SS]E\0" where E is a char that indicates the error
  • "[TIME ND]\0" if time is not set
    Returns
    char* Array that will contain the timestamp string.
